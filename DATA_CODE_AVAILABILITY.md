# Data and Code Availability Statement

## Manuscript-ready statement

**Data and code availability.** The study uses two operating-room-duration-calibrated semi-synthetic benchmark families and one synthetic stress family. The reproducibility package contains project code, algorithm and solver configurations, source-specific preprocessing scripts, generated metadata, selected-schedule hash audits, derived result tables, and scripts that reproduce the final figures and manuscript robustness analyses. GermanOR frozen duration pools are included with CC-BY-4.0 attribution. The Mannino raw workbook and Mannino-derived duration pools are not redistributed because an explicit licence authorizing redistribution or sublicensing was not identified; for Mannino, the package provides source retrieval instructions, deterministic preprocessing and pool-generation code, and SHA-256 manifests for verification. A public repository URL and archived Zenodo version DOI will be inserted after the release is frozen.

## Compact journal-system version

Code, configurations, generated metadata, derived outputs, GermanOR frozen pools, and reproducibility audits are archived in a public repository and Zenodo release to be inserted after archival. Mannino source data are retrieved directly from SINTEF; derived Mannino duration pools are verified through deposited preprocessing code and SHA-256 manifests rather than redistributed.
