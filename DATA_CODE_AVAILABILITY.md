# Data and Code Availability Statement

## Manuscript-ready statement

**Data and code availability.** The study uses two operating-room-duration-calibrated semi-synthetic benchmark families and one synthetic stress family. Project code, algorithm and solver configurations, source-specific preprocessing scripts, generated metadata, selected-schedule hash audits, derived result tables, and scripts reproducing the final figures and manuscript robustness analyses are available at https://github.com/JiamingHu726/or-icu-ward-recourse-guidance. GermanOR frozen duration pools are included with CC-BY-4.0 attribution. The Mannino raw workbook and Mannino-derived duration pools are not redistributed because an explicit licence authorizing redistribution or sublicensing was not identified; retrieval instructions, deterministic preprocessing and pool-generation code, and SHA-256 verification manifests are provided. A versioned Zenodo archive DOI: 10.5281/zenodo.20892041.

## Compact journal-system version

Code, configurations, generated metadata, derived outputs, GermanOR frozen pools, and reproducibility audits are archived in a public repository and Zenodo release to be inserted after archival. Mannino source data are retrieved directly from SINTEF; derived Mannino duration pools are verified through deposited preprocessing code and SHA-256 manifests rather than redistributed.
