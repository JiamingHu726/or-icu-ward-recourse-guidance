# Public data access policy

## GermanOR

The repository includes frozen GermanOR duration pools with their upstream CC-BY-4.0 attribution. The pool artifacts are derived from the German OR Benchmarking source and are included only under that source's stated attribution terms. See `data/derived/frozen_calibration/GermanOR/ATTRIBUTION.md`.

## Mannino

The Mannino raw workbook and Mannino-derived pools are not redistributed. The package instead provides public preprocessing/generation code, retrieval notes, and SHA-256 manifests to support independent regeneration from the provider-hosted source.

## Synthetic family

The synthetic stress family is generated by this repository's software and configuration files. Generated aggregate results used in the manuscript are included.

## Exclusions

No raw Mannino workbook, private source copy, Gurobi licence, credential, or machine-specific raw experiment directory is part of this release.
